Generators are divided into two groups.

  • The first group has a Context level.
  • The second one is a View-Model level.

Context level code is generated once and shared by other parts of the application.
The View-Model level code must be regenerated for each view you select. That is, each view you select must have its own copy of the generated code. The View-Model level code generated for one view cannot be used by another view. For example, the product attributes input form is View-Model level code

Each generator has a title which starts with 5 digits.
If the first two digits are zeros it is Context level generator for Wizards
If one the first two digits is non zero it is View-Model level generator for Wizards

The currently selected generator is a Context level code generator.

CLI-related information
Inside your Angular project
1. Run the command

    
            ng g component app
2. It will create the folder app and inside create the set of files app.component.*
3. In Visual Studio right-click created fordel and generate the component-body It updates app.component.css only.

Project-related information
Step 0: Do not run the command described under CLI-related information. It's for information purpose only and makes you hint about the type of component you are going to generate
Step 1: Inside folder src/app create the folder 'shared' (You can choose any folder name you like: 'myshared' or 'mycode' and so on)
Step 2: Inside folder of step 1 (recommended 'shared') create the folder '/src/app/' (You can choose any folder name you like: 'mymodules' or 'fooooo' and so on)
Step 4: In Visual Studio right-click the '/src/app/'-folder that you created in step 2 and generate the following items

  • 00210-app.component.css
  • 00210-app.component.html
  • 00210-app.component.ts
Don't forget to click save-button. Even if the file is empty!!! It saves the code into the file and adds some information into repository of the Wizards. Repository information is actively used by all other generators that you will run later.

00210-app.component.ts is root Application.
